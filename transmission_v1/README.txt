# crowd_simulation

Projet d'informatique P2 du semestre 4

Objectif : créer une simulation de propagation d'un virus ou d'un sentiment dans une foule

Etape 1 : Modéliser un mouvement de foule à l'aide de l'algorithme des BOIDS
		- se rassembler
		- s'éloigner
		- se suivre

-> foule de petits dinosaures Wanis

Etape 2 : Progager quelque chose à l'aide de l'alogorithme SIR 
		- Suceptible
		- Infected
		- Recovered
